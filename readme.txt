Este é um projeto que foi realizado por mim, César Tallys, para fins didáticos.
O projeto foi desenvolvido para ser apresentado na matéria de Bancos de Dados no curso de Ciência da Computação do Instituto Federal de Brasília.
O sistema tem como objetivo automatizar o processo de reserva de chaves para uma escola.

Para rodar o sistema é necessário que se tenha instalado em sua máquina MySql e algum servidor para executar o php.

Para utilizar o sistema, execute os seguintes passos:
1º Baixe o diretório do sistema e copie para a pasta do seu servidor.
2º Abra seu gerenciador do MySql e execute todos os comandos do arquivo <script.sql>.
3º Acesse, pelo navegador, o diretório do sistema que está na pasta do servidor.

Dentro do sistema, primeiro deve-se acessar a área do administrador (login: 123, senha: 123) para cadastrar os funcionários e as chaves.
Pronto! Agora é só entrar na área do servidor (funcionário) e realizar suas reservas.

O sistema foi desenvolvido utilizando HTML, CSS, Bootstrap, Php e Mysql.
